# Auto-Deploy Setup

Since I can't access Vercel CLI authentication, I need you to create a Deploy Hook (one-time setup, 30 seconds):

## Quick Setup:

1. Go to: https://vercel.com/futuretechtoday77s-projects/gcmodal-api/settings/git
2. Scroll to **"Deploy Hooks"**
3. Click **"Create Hook"**
4. Name: `Auto Deploy`
5. Branch: `main` (or `master`)
6. Click **"Create Hook"**
7. **Copy the URL** it gives you (looks like: `https://api.vercel.com/v1/integrations/deploy/...`)
8. Send me that URL

Then I can deploy anytime with one command!

**OR even simpler - just do this once:**

Go to https://vercel.com/futuretechtoday77s-projects/gcmodal-api

Click the blue "Visit" button dropdown → "Redeploy" → Confirm

Done! That's literally all you need to do. One click.
