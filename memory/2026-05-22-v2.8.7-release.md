# MV Popup Manager v2.8.7 - Production Release

**Date:** 2026-05-22  
**Version:** 2.8.7  
**Status:** Production

## Overview

Stable release with working popup functionality and cookie-based restrictions.

## What's Working ✅

### Button Popups
- **NO cookie restriction** - Always show when clicked
- Works on all pages
- No limitations

### Exit Intent Popups
- **7-day cookie restriction**
- Triggers when mouse moves to top 50px of viewport
- Shows once per 7 days per popup ID

### Time Delay Popups  
- **7-day cookie restriction**
- Default delay: 3 minutes (180 seconds)
- Shows once per 7 days per popup ID

### API
- CORS headers properly configured
- Returns correct popup data
- No errors
