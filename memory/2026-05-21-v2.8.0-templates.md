# MV Popup Manager v2.8.0 - Template System Release

**Date:** 2026-05-21  
**Version:** 2.8.0  
**Status:** Production

## Overview

Major release introducing a comprehensive template system with 9 pre-designed popup layouts, mobile-first responsive design, and extensive customization options.

## New Features

### Template System (9 Templates)

**Minimal Templates:**
1. **Clean Gradient** - Simple gradient background with centered form
2. **Ultra Minimal** - Text only, single field for maximum conversion

**Image Templates:**
3. **Split Screen** - Image on left, form on right with full image display
4. **Full Background** - Full-bleed background image with overlay form
5. **Full Background Tall** - Taller version with more vertical space
6. **Full Background Wide** - Wider desktop view (650px)
7. **Full Background Compact** - Minimal height (250px)

**Product Templates:**
8. **Lead Magnet** - Product image with download form

**Personal Templates:**
9. **Personal Consultation** - Professional photo with floating avatar, chat-style mobile

### Mobile-First Design

- Default preview is now mobile (320px)
- Desktop toggle available
- All templates responsive
- Touch-friendly inputs

### Customization Options

**All Templates:**
- Color Theme (12 options including 4 high-contrast professional themes)
- Button Color (8 preset colors)
- Popup Height (Compact, Standard, Tall)
- Custom Text Colors (headline/subheadline)
- Trust Text (customizable with on/off toggle)

**Full Background Specific:**
- Overlay toggle (on/off)
- Overlay color picker
- Overlay opacity slider (10-90%)

**Personal Consultation Specific:**
- Avatar image upload/URL
- Avatar position (left/right)
- Chat message text

### Color Themes

**New High-Contrast Professional Themes:**
- Professional (White/Dark)
- Professional Blue
- Dark Mode
- Clean Minimal

**Original Pastel Themes:**
- Purple, Blue, Green, Red, Orange, Pink, Dark, White

### Image Handling Improvements

- All images use `object-fit: contain` (no cropping)
- Full Background shows image on mobile
- Split Screen displays full image at 50% width
- Lead Magnet has flexible image sizing

## Technical Changes

### New Files
- `/app/admin/components/templates/template-config.js` - Template definitions
- `/app/admin/components/templates/TemplateSelector.js` - Template picker UI
- `/app/admin/components/templates/PopupPreview.js` - Template rendering

### Modified Files
- `/app/admin/popup/edit/page.js` - Added template tab, design options
- `/app/admin/page.js` - Updated to work with template system

### New State Fields
```javascript
popup: {
  template: 'clean-gradient',           // Template ID
  avatarUrl: '',                        // Personal consultation
  avatarPosition: 'bottom-left',        // Avatar position
  chatMessage: 'Want to have a free...', // Chat bubble text
  trustText: 'We respect your email...', // Trust message
  showTrustText: true,                  // Show/hide trust text
  showOverlay: false,                   // Full background overlay
  overlayColor: '#000000',              // Overlay color
  overlayOpacity: 50,                   // Overlay opacity
  buttonColor: '#3b82f6',               // Custom button color
  popupHeight: 'standard',              // Compact/Standard/Tall
  useCustomTextColors: false,           // Enable custom colors
  headlineColor: '#000000',             // Custom headline color
  subheadlineColor: '#666666'           // Custom subheadline color
}
```

## Deployment Notes

**Current Process:**
1. Save popup in admin (saves to database)
2. Manually add to `/app/api/popups/route.js` and `/app/api/admin/popups/route.js`
3. Commit and push
4. Vercel auto-deploys in 30-60 seconds

**Coming in v2.9.0:**
- Auto-deploy feature
- Saving in admin will automatically update config files and deploy

## User Experience

### Template Selection
1. Click "Template" tab in edit page
2. Browse by category (All, Minimal, With Image, Product, Personal)
3. Click template to preview
4. See live preview update immediately

### Design Customization
1. Click "Design" tab
2. Adjust color theme, button color, height
3. Toggle trust text, overlay settings
4. Set custom text colors if needed
5. All changes reflect in live preview
